Spend Wisely – AI Powered Personal Finance Assistant

Spend Wisely is an AI-driven personal finance management platform that helps users track expenses, manage budgets, analyze spending behavior, and receive intelligent financial insights.

Features

• Financial dashboard with net worth, income, expenses and savings rate
• Expense and income tracking
• Voice-based transaction logging
• Receipt scanning
• Category-based budgeting
• AI-powered financial insights
• Smart savings recommendations
• Investment planning with SIP suggestions
• Portfolio diversification suggestions
• AI finance assistant chatbot

Tech Stack:

Frontend:
React
TypeScript
Tailwind CSS

Backend:
Serverless APIs

Database:
Cloud database

Deployment:
Vercel

Installation

Clone the repository

git clone https://github.com/yourusername/spend-wisely.git

Install dependencies

npm install

Run the development server

npm run dev
